# Art Folder

Put your artwork images here!

## Supported Formats
- JPG/JPEG
- PNG
- GIF
- WebP

## Recommended Image Size
- Width: 600-1200px
- Height: 600-1200px (square or vertical works best)

## File Naming
Name your files descriptively:
- Good: `sunset-painting-2024.jpg`
- Good: `digital-portrait.png`
- Avoid: `IMG_1234.jpg`

## Adding to Your Gallery

### Manual Method:
1. Put your image in this folder
2. Open `play.html`
3. Copy the template in the HTML comments
4. Replace `your-image.jpg` with your filename
5. Fill in title and description

### Automatic Method (if Instagram setup works later):
The `update_art_gallery.py` script will fetch from Instagram and handle everything automatically!
