ItEra Studio

AI-powered reference search for artists. It understands creative intent and returns relevant real-world sources instead of generated imagery.

Back to Work

Overview

ItEra Studio is an intent-aware reference search engine for artists. It interprets creative intent, searches real art and image databases, and returns credited references with source links and license-aware filtering.

The project began as early prompt experiments, then shifted to a clearer direction: AI for search, understanding, and discovery, not generation.

Problem

  • Artists often know the visual direction they want but struggle to convert it into effective queries.
  • Keyword-only search is blunt and imprecise for mood, composition, style, and medium.
  • Reference discovery is fragmented across disconnected platforms, each with different metadata quality.
  • Many AI tools generate images instead of helping artists study real references and source material.
  • Most discovery tools are not personalized to an artist's intent, preferences, or workflow context.

My Role

  • Defined the end-to-end search flow: user input -> query interpretation -> multi-API retrieval -> ranked results.
  • Designed ranking and filtering logic around relevance, image validity, attribution quality, and licensing.
  • Mapped API routing strategies by intent and medium (for example, museum collections vs. photography sources).
  • Designed the discovery UX so artists can scan, compare, and save references with clear source context.
  • Shaped product direction through iterative research, prototyping, and content testing.

Research Foundations

  • Intent-aware search is essential because artists describe ideas conceptually, not with rigid keywords.
  • Personalization improves workflow fit by adapting results to preferred subjects, styles, and mediums.
  • Real references are critical for learning technique, composition, and visual analysis from authentic sources.
  • Generative outputs can bypass observation and source literacy, which weakens long-term artistic development.
  • Strong discovery systems need both relevance and trust: valid images, credit, links, and licensing context.

Design Goals

Intent-Aware Search

  • Understand what the user means, not only what they type.
  • Expand ambiguous input into richer search-ready query sets.

Multi-Source Aggregation

  • Pull from verified datasets including Met, AIC, Smithsonian, Unsplash, and Wikimedia.
  • Route queries to the most relevant sources by content type and intent.

Relevance Ranking + Deduplication

  • Score results by semantic relevance, metadata quality, and image validity.
  • Remove duplicates across providers to keep results clean and diverse.

Attribution & Ethical Sourcing

  • Return credited references with artist, source, and direct links.
  • Apply license-aware filtering so results are usable and transparent.

Personalized Discovery

  • Adapt ranking using user preferences, saved references, and search history.
  • Support creative exploration while maintaining relevance and variety.

System Design

  • Query interpretation + expansion: Parse user intent, then expand to related visual terms, movements, materials, and contexts.
  • API routing logic: Route portrait and fine-art queries to museum datasets, and photography-heavy intents to sources like Unsplash and Wikimedia.
  • Ranking + scoring: Blend intent match, metadata completeness, source trust, and image quality into a single relevance score.
  • Deduplication: Merge repeated works across APIs using title, artist, and visual similarity checks to reduce clutter.
  • Metadata requirements: Each result must include image, artist credit, title (when available), source link, and license details.

Future Development

  • Improve ranking with stronger relevance weighting and feedback-driven tuning.
  • Expand user preference modeling for better personalization over time.
  • Advance semantic matching to capture nuanced artistic intent.
  • Add more datasets, with emphasis on diverse and non-Western collections.

Reflection

ItEra Studio clarified the role I want AI to play in creative tools: not generating finished work, but improving discovery. By helping artists find real, credited references faster, AI can augment learning, strengthen visual research, and support creativity without replacing it.
