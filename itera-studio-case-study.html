ItEra Studio

A reference search tool for artists that helps users find high-quality visual sources by interpreting creative intent instead of generating imagery.

Back to Work Open Functional Prototype

Overview

ItEra Studio is an evolving reference search system for artists. It is being designed to interpret creative intent, search across real art and image databases, and return credited references with source links and license-aware filtering.

The project originally began during my freshman year as a simple Java-based quiz that generated art prompts. Over time, it evolved through multiple iterations, first into AI-assisted prompting, then into critique-based exploration, and ultimately into a system focused on reference discovery.

Its future development includes integrating an appropriate LLM to support intent interpretation, query expansion, and more personalized search without replacing the artist’s creative process.

Prototype: Built as an early concept with a live interface direction and a backend plan centered on multi-source retrieval, ranking, and credited results.

Problem

  • Artists often know the visual direction they want but struggle to translate it into effective queries.
  • Keyword-based search is too rigid for concepts like mood, composition, style, and medium.
  • Reference discovery is fragmented across platforms with inconsistent metadata, attribution, and source quality.
  • Many AI tools prioritize generating images instead of helping artists study real references.
  • There is significant skepticism within the art community toward AI, especially around ethics, authorship, and trust.

My Role

  • Led the full evolution of the project from early prompt generation in Java to a research-driven product direction.
  • Designed the end-to-end system: user input to intent interpretation to multi-source retrieval to ranked results.
  • Conducted user research through surveys, community outreach, and small-scale testing.
  • Defined ranking, filtering, and attribution logic to prioritize relevance, trust, and usability.
  • Iterated on product direction based on findings, shifting from critique systems to reference-based discovery.

Key Turning Point

Early iterations of this project explored AI-generated prompts and critique systems. However, through user surveys and community feedback, I found that many artists were skeptical of AI critique tools due to concerns around accuracy, ethics, and legitimacy.

While some users found value in AI feedback, there was stronger interest in tools that helped them find and study real references. Based on this, I shifted the project toward reference discovery: using AI to interpret intent and surface relevant, credited work rather than generate or evaluate it.

Design Goals

Intent-Aware Search

  • Interpret conceptual input, not just keywords.
  • Expand queries into richer, search-ready terms.

Multi-Source Aggregation

  • Pull from verified datasets such as the Met, AIC, Smithsonian, Unsplash, and Wikimedia.
  • Route queries based on intent and content type.

Relevance Ranking + Deduplication

  • Rank results using relevance, metadata quality, and image validity.
  • Remove duplicate works across sources to keep results cleaner and more useful.

Attribution & Ethical Sourcing

  • Return artist credit, source, and direct links.
  • Filter results based on licensing and usability.

Personalized Discovery

  • Adapt results based on preferences and behavior.
  • Support exploration while maintaining relevance.

Example Output

A user might search for something like "dynamic anime fight pose with dramatic lighting". The system would then interpret that intent and expand it into related concepts such as pose, motion, foreshortening, composition, and lighting conditions.

  • User input: dynamic anime fight pose with dramatic lighting
  • Expanded search direction: dynamic pose, action composition, foreshortening, strong contrast, cinematic lighting
  • Returned results: relevant references from curated art and image datasets
  • Metadata shown: artist, source, direct link, and license information

System Design

  • Query interpretation + expansion: The long-term system is designed to parse user intent and translate conceptual language into related visual terms, styles, and contexts.
  • API routing logic: Queries are routed to appropriate datasets depending on content type, such as fine art, photography, or illustration, so users get more relevant results.
  • Ranking + scoring: Results are evaluated using intent match, metadata quality, source trust, and image validity to help the most useful references appear first.
  • Deduplication: Repeated works are removed using title, artist, and similarity checks to reduce clutter.
  • Metadata requirements: Each result should include an image, artist, title, source link, and license information so the user can evaluate and trust what they are seeing.

Evaluation

I conducted surveys and gathered feedback through online communities, as well as small-scale user testing. Public responses showed strong skepticism toward AI critique tools, while private testing (n = 4) indicated moderate usefulness when feedback was personalized.

These findings supported the shift toward reference-based discovery, where users showed greater trust and willingness to engage.

Future Development

  • Integrate an appropriate LLM to support stronger intent interpretation and query expansion.
  • Use the model to better translate conceptual artistic language into structured search terms across datasets.
  • Improve ranking through feedback-driven tuning and stronger relevance weighting.
  • Expand personalization using saved references, preferences, and user behavior over time.
  • Refine semantic matching for more nuanced artistic intent.
  • Expand datasets with more diverse and non-Western sources.

Reflection

This project reflects a longer evolution in how I think about AI in creative tools, from early prompt generation to more intentional, research-driven design decisions.

I found that the most effective role for AI is not generating outcomes, but supporting discovery and learning. Designing ItEra Studio reinforced the importance of trust, ethics, and alignment with user workflows, especially in creative domains.

More broadly, this project demonstrates how AI can support creative workflows without replacing them, focusing on trust, usability, and real artistic learning.
