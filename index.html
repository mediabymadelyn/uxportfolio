Selected projects

Product, UX, and interface work shaped through research, prototyping, and implementation.

Fall 2024 - Present Product Design + Creative Technology

ItEra Studio

A reference search tool for artists that helps users find real, useful visual sources based on what they are trying to make. I shaped the product direction through research and designed the system around trust, intent, and ethical sourcing.

View Case Study

Spring 2025 Product Design + Frontend

Oxy Biochemistry Database

A research database designed for Occidental’s Biochemistry Department. I focused on frontend implementation and interaction design to make lab workflows clearer and more usable.

View Case Study

Spring 2026 AI + Learning Technologies

Mise-en-Lens

A Next.js web app that teaches film theory through a user’s own taste. Users input their Top 4 films and the system uses an LLM to generate personalized analysis connecting their films to broader concepts in cinematography, directing, societal context, and genre.

View Case Study

Spring 2026 Python + Systems Design

Vibe.py

A personal Python terminal app for music discovery and movie roulette. Built from scratch with Spotify OAuth, adaptive audio feature scoring, and a persistent preference profile that evolves with feedback.
